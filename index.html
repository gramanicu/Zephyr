Privacy
You don't like giving away information?

Don't worry. The site keeps very little information on the users. We use a "kind of" peer-to-peer connection: you send the message to the server, the server sends it forward. Without saving the text or images in a database. When you close the window, all the data is lost.

Optimised for mobile

A lot of work was put into this application to make sure it looks good anywhere. Minimalistic design, optimisations, everything I could do to give any user the best experience he can have.
